Selected Research Projects

Optimal spectrum estimation from samples via moment matching   [Poster][Slides]

Tiancheng Yu, Jiantao Jiao, Yanjun Han Tsachy Weissman

Estimaing the eigenvalues of the covariance matrix for multivariate random variables from samll sample size but high dimensionality is a challenging task in modern data science, with applications in dimension reduction and latent variable inference. Based on the moment matching method proposed in [Weihao Kong, Gregory Valiant 2016], we improve the theoretical upper bound and empirical performance, establish the intuition when moment matching is optimal and prove the optimality by constructing a mini-max lower bound.

Indoor localization with landmark learning: Theory and Algorithm  

Tiancheng Yu, Feng Ge, Yuan Shen

Indoor localization with landmark learning is a substitute to the fingerprint method with manual calibration via crowdsourcing. In this project, we develop a algorithm achieves the minimum error in the Bayesian sense. We implement it and get promising results in Rohm Building, Tsinghua University. We derive a compact upper bound for limit accuracy and convergence rate, using Fisher information and algebriac graph theory.

Preprocessing to achieve more robust neural network  

Tiancheng Yu, Dimitris Papailiopoulos

Although neural networks have achieve tremendous success in various domains, it is easy to be fooled by adversarial sample. In this project, we develop a preprocessing procedure by adding additional noise. We derived an upper bound for the robustness when using SGD for training.
